# Python file integrity checker
This repository holds a basic file integrity checker using the SHA-256 algorithm
## Objective 
The main objective was to create a python file that could verify whether a file that you want to download from a download link has been tampered with by comparing its SHA-256 agaisnt a known trusted hash. Additonally I aimed to understand what file integrity checking is, how it can be used to detect changes in files and why this is important in the world of cyber security.
## Skills learnt
- Python coding
- Python librarys
- SHA-256 Database
## Tools Used
- VS code
## Steps
### Prerequsities 
None, the haslib library is built into python
### Run the file
Open the file in your IDE, on line 3 fill in the hash you are expecting, and on line 5 input the exe file. After you should be able to compare the hashs.
