# Password Hashing
This repository holds a basic hashchecker for passwords.
## Objective 
The main objective was to create a python file that could hash a password and verify it without storing it in plain text. Alongside this my aim at the end of the project was to understand what hashing is, how it is used to protect sensitive information in data bases and why stroing passwords in plain text can be a security risk.
## Skills learnt
- Python coding
- Python librarys
- SHA-256 Database
## Tools Used
- VS code
## Steps
### Prerequsities 
None, the haslib library is built into python
### Run the file
Open the file in your IDE, change the password on line 10, run the file and compare the hashs displayed.
